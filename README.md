# 💳 Loan Credit Forecasting

A machine learning project developed in Python to predict loan approval outcomes based on applicant information and financial characteristics.

This project uses data preprocessing, feature engineering, and Logistic Regression to analyze customer profiles and forecast loan eligibility. The goal is to assist financial institutions in making data-driven lending decisions.

---

## 📌 Features

* Loan approval prediction
* Credit risk assessment
* Data cleaning and preprocessing
* Missing value handling
* Categorical data encoding
* Correlation analysis and visualization
* Logistic Regression classification model
* Model performance evaluation

## 📊 Data Processing Pipeline

1. Load loan application dataset.
2. Remove unnecessary columns.
3. Encode categorical variables.
4. Handle missing values.
5. Perform exploratory data analysis.
6. Generate correlation heatmaps and visualizations.
7. Split data into training and testing sets.
8. Train a Logistic Regression model.
9. Evaluate prediction accuracy.

---

## 🤖 Machine Learning Model

**Algorithm Used:**

* Logistic Regression

**Target Variable:**

* Loan_Status

**Input Features Include:**

* Gender
* Marital Status
* Dependents
* Education
* Self Employment Status
* Applicant Income
* Coapplicant Income
* Loan Amount
* Loan Term
* Credit History
* Property Area
